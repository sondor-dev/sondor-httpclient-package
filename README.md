# Sondor HTTP Client
![Build Status](https://dev.azure.com/sondortechnology/Sondor%20Infrastructure/_apis/build/status%2Fsondor-dev.sondor-logging-loki-package?repoName=sondor-dev%2Fsondor-httpclient-package&branchName=master) ![NuGet Downloads](https://img.shields.io/nuget/dt/Sondor.HttpClient)

The Sondor HTTP Client package, provides common functionality and extensions to simplify the use of HTTP clients.

## Install Options
Install via NuGet
```cli
Install-Package Sondor.HttpClient
```
Install via .NET Core command line
```cli
dotnet add package Sondor.HttpClient
```
